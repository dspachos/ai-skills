# AI Skills

A collection of [Agent Skills](https://agentskills.io/home) for AI agents and CLI tools.

Agent Skills are folders of instructions, scripts, and resources that agents can discover and use to do things more accurately and efficiently. These skills extend agent capabilities with specialized knowledge, workflows, and tool integrations.

## Available Skills

### Drush CLI
Drupal command-line operations via Drush for system inspection, entity management, cache clearing, configuration, database operations, and module/theme management.

- System status checks
- Entity inspection (nodes, users, taxonomy)
- Configuration management and search
- Cache clearing strategies
- User management
- Database operations

### GitHub CLI (gh)
Work with GitHub repositories, pull requests, issues, projects, and workflows using the GitHub CLI.

- Pull request creation, review, and management
- Issue tracking and management
- GitHub Projects
- Workflow and Actions monitoring
- Repository operations

## Using These Skills

These skills are compatible with Agent Skills-enabled tools including:

- [OpenCode](https://opencode.ai/)
- [Claude Code](https://claude.ai/code)
- [Cursor](https://cursor.com/)
- [Gemini CLI](https://geminicli.com)
- And more

### Installation

The installation process depends on your agent tool. Most Agent Skills-compatible tools support the `.skill` format.

## About Agent Skills

Agent Skills provide:

- **Domain expertise** - Package specialized knowledge into reusable instructions
- **New capabilities** - Give agents new abilities through tool integrations
- **Repeatable workflows** - Turn multi-step tasks into consistent workflows
- **Interoperability** - Use the same skill across different agent products

Learn more at [agentskills.io](https://agentskills.io/home).
